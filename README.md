# courier-management-project

DBMS Project on courier management system


For using on your system:

1. Clone this Repo on your system.
2. go in the repo: 
  `$ cd repository`
3. Install the dependencies from the requirements file: 
  `$ pip install -r requirements.txt`
4. run frontend.py file: 
  `$ python3 frontend.py`
  This will start the server in debug mode.
